Skill Hub

Skill Hub is a web-based peer-to-peer platform designed to enable users to exchange skills and collaborate based on shared interests. The platform allows users to create profiles, list their skills, and connect with others for mutual learning.

Features

  • User authentication and login system
  • Profile creation and skill listing
  • Skill matching between users
  • Interactive and user-friendly UI
  • Database integration for storing user data
